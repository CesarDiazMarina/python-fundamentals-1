## 📌 Descripción
La aplicación integra los conceptos vistos en el módulo: variables,
estructuras de datos, control de flujo, funciones, programación funcional y
programación orientada a objetos (POO), a través de 4 ejercicios prácticos.

## 🧭 Estructura de la app
- **Home**: presentación del proyecto.
- **Ejercicio 1**: flujo de caja con listas.
- **Ejercicio 2**: registro de productos con NumPy y DataFrame.
- **Ejercicio 3**: uso de una función externa (`libreria_funciones_proyecto1.py`).
- **Ejercicio 4**: uso de una clase externa con operaciones CRUD (`libreria_clases_proyecto1.py`).
